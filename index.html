Skills

I use the following in my work

  • HTML5

    HTML (HyperText Markup Language) is the standard markup language used to create and design documents on the World Wide Web. It structures web pages by using a series of elements and tags, which define the content and layout.

  • CSS3

    CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. CSS controls the layout of web pages, including the design, colors, fonts, and spacing.

  • Java Script

    JavaScript (JS) is a high-level, dynamic programming language commonly used in web development. It enables interactive and dynamic content on websites, such as animations, form validation, and user interactions.

  • jQuery

    jQuery is a fast, small, and feature-rich JavaScript library. It simplifies tasks like HTML document traversal and manipulation, event handling, animation, and Ajax interactions with an easy-to-use API that works across a multitude of browsers.

  • React

    React is a popular JavaScript library for building user interfaces, primarily for single-page applications. Developed and maintained by Facebook, React allows developers to create reusable UI components that manage their own state.

  • Node.js

    Node.js is a runtime environment that allows developers to run JavaScript on the server side. Built on Chrome's V8 JavaScript engine, it is designed for building scalable network applications.

  • Gulp

    Gulp is a task runner and build system for automating repetitive tasks in web development. It is built on Node.js and uses a code-over-configuration approach to define tasks, making it easy to read and write.

  • VS Code

    VSCode features a powerful integrated development environment (IDE) with built-in support for debugging, version control, and task running.
